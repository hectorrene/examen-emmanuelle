# si mañana CETYS cambia de proveedor de catálogo a uno con una interfaz completamente diferente, ¿cuánto código habría que modificar? ¿Por qué?

- En caso de que se cambie de proveedor no se tendría que cambiar nada del código principal o de cómo se manda al catalogo de CETYS, ya que solo se tendría que crear un nuevo adaptador hacia el nuevo sistema / proveedor y descartar el obsoleto.
